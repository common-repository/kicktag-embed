Kicktag plugin allows you to embed kicktag videos easily on your blog

== Description ==

The kicktag plugin provides a shortcode for blogger to quickly post kicktag video in their wordpress blogs.

== Installation ==

Upload the kicktag_embed plugin to your blog, Activate it.

1, 2, 3: You're done!

== Frequently Asked Questions ==

= How to use the shortcode? =

Basic Syntax: [kicktag kicked_video_id="<kicktag video id from site>" host="<kicktag video hosting site>"]
